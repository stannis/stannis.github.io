当git status命令出现红字字体时，先用命令
git checkout --
然后删掉远程库上的某文件,命令如下
git rm _post/2014-05-22-中文测试.md
再git status比较
此时 出现绿字“deleted "_posts/2014-05-10-\344\270\255\346\226\207.md"
这时就可以推送更新了
git commit -m "del"
git push origin master
